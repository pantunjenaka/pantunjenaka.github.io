29. sebuah tangga panjangnya 4 meter bersandar pada tembok jarak ujung tembok bawah tangga dengan tembok 3 meter berapa tinggi tembok yang di capai ujung tembok​


Jawaban

[tex] \sqrt{7} [/tex] m

Penjelasan

Bentuk tangga seperti yang digambar.

a = sisi siku² (terpendek)

b = sisi siku²

c = sisi miring (terpanjang)

b = 3 meter

c = 4 meter

a = ?

a² = c² - b²

a² =.4² - 3²

a² = 16 - 9

a² = 7

[tex]a = \sqrt{7} \: m[/tex]

36. sebuah tangga disandarkan pada tembok jika jaraknkaki tangga dan tembok adalah1,8dan tinggi tembok yang dicapai tangga adalah2,4m.kemiringan tangga terhadap tembok adalah​


Diketahui :

alas = 1,8 meter

tinggi = 2,4 meter

Ditanya :

kemiringan?

Rumus : [tex]a^{2} + b^{2} = c^{2}[/tex]

Jawab :

a = alas

b = tinggi

c = kemiringan

[tex]1,8^{2} +2.4^{2} = c^{2} \\3,24 + 5.76 = c^{2}\\9 = c^{2}\\c = \sqrt{9}\\ c = 3\\[/tex]

Jadi, kemiringan tangga terhadap tembok adalah 3 meter
