4. Harga 3Kg salak dan 2kg kedondong rp.19.500,sedangkan harga 2kg salak dan 3kg kedondong rp.20.000,Tentukan dan selesaikan Model matematika nya,tentukan Juga harga 1 kg salak,1 kg kedondong,dan harga 2kg salak dan 3Kg kedondong Mohon bantuannya ya


Salak : x
kedondong : y

* model matematika :
3x + 2y = 19.500
2x + 3y = 20.000
* Penyelesaian :
eliminasi 'x'
3x + 2y = 19.500 [x2] 6x + 4y = 39.000
2x + 3y = 20.000 [x3] 6x + 9y = 60.000
_______________-
-5y = - 21.000
y = 4.200
Harga 1kg kedondong Rp. 4.200

Substitusi (y=4.200) ke pers 1
3x + 2y = 19.500
3x + 2(4.200) = 19.500
3x + 8.400 = 19.500
3x = 19.500 - 8.400
3x = 11.100
x = 11.100/3
x = 3.700
Harga 1 kg Salak Rp. 3.700
* Harga 2kg salak dan 3kg kedondong
2x + 2y
=2(3.700) + 2(4.200)
=7.400 + 8.400
=15.800
Jadi harga 2kg salak dan 3kg kedondong adalh Rp. 15.800
