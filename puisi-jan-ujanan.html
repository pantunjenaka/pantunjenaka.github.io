10. kenapa tejadinya guntur atau patir saat mau ujan


Kejadian petir dan kilat merupakan gejala listrik statis. Dalam ilmu fisika sekolah, petir berasal dari pemuaian udara yang cepat akibat dilalui oleh loncatan bunga api listrik. Loncatan bunga api listrik yang lebih besar sering terjadi pada saat awan mendung atau sedang terjadinya hujan.
Sekumpulan awan hitam dapat memperoleh muatan listrik yang sangat kuat karena pergesekan dengan udara. Hal ini menimbulkan bunga api listrik yang sangat besar dari kelompok awan ke  awan lain yang muatannya berlawanan. Loncatan bunga api listrik kita kenal dengan kilat.


Loncatan bunga api listrik menimbulkan panas yang sangat besar sehingga menyebabkan udara yang dilaluinya memuai dengan cepat. Pemuaian yang cepat dan tiba-tiba menimbulkan suara keras  yang dikenal dengan guruh atau guntur.

32. buat lah puisi yng sedikit panjang dan ubah puisi itu menjadi prosadan Jan lupa isi judul puisi​


Judul : Cahaya dalam Gelap

Puisi:

Malam ini, kurasakan dingin menusuk tulang

Sepi mengiringi langkah-langkahku

Ku berjalan dalam gelap, mencari cahaya

Di ujung jalan, kulihat api unggun yang membara

Kupandang api itu dengan kagum, seakan menghipnotis diriku

Malam semakin gelap, tapi api itu semakin terang

Ku menghampiri api itu, merasakan hangatnya

Terasa seperti pulang ke pelukan ibu

Aku merenung, sejenak terlena dalam api itu

Namun kemudian aku tersadar, api itu hanya ilusi semu

Kembali kutatap malam yang gelap, tetap berjalan mencari cahaya

Prosa:

Malam ini, aku merasakan dingin menusuk tulangku. Sepi mengiringi langkah-langkahku. Aku berjalan dalam gelap, mencari cahaya yang mungkin bisa menerangi jalanku. Di ujung jalan, aku melihat api unggun yang membara. Kulihatnya dengan kagum, seakan api itu memikatku. Meski malam semakin gelap, tapi api itu semakin terang. Aku menghampiri api itu, merasakan hangatnya yang terasa seperti pelukan ibu. Sejenak aku merenung, terlena dalam kehangatan api itu. Namun kemudian aku tersadar, bahwa api itu hanya ilusi semu yang tidak akan membawaku ke mana-mana. Kembali kutatap malam yang gelap, tetap berjalan mencari cahaya yang mungkin bisa membawaku ke tempat yang benar.
