33. sebuah toko membeli 3 rim kertas HVS untuk dijual eceran. Harga pembelian 1 rim kertas HVS Rp. 30.000. Jika habis terjual keuntungan yang diperoleh Rp. 60.000. Tentukan : a.) harga jual seluruh kertas HVSb.) harga jual kertas HVS per lembar​


Diket :

1 rim = 500 lembar

3 rim = 1500 lembar

1 rim = 40.000

3 rim = 120.000

Harga beli selembar = 40.000/500 = Rp80

Untung total = Rp60.000

Untung perlembar = 60.000/1500 = Rp40

Dit + jwb :

a. Harga jual seluruh kertas HVS

Harga beli + keuntungan

120.000 + 60.000

Jdi Rp180.000

b. Harga jual Kertas Hvs perlembar

Harga beli selembar + keuntungan perlembar

80 + 40

Jdi harga jual selembar adalah Rp120

Penjelasan dengan langkah-langkah:

diketahui:

1 rim = 500 lembar

3 rim = 500x3 = 1500 lembar

harga beli 1 rim = Rp 30.000

jika 3 rim = Rp 30.000 x 3 = Rp 90.000

keuntungan = Rp 60.000

ditanyakan:

a. harga jual seluruh kertas hvs = harga beli + keuntungan = Rp 90.000 + Rp 60.000

= Rp 150.000

b. harga jual kertas hvs per lembar = Rp 150.000/1500 lembar = Rp 100
