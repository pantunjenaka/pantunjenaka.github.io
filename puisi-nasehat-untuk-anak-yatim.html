Uang untuk anak yatim yaitu Rp. 3.750.000

Langkah² ada di gambar

-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•-•

Dik = Uang awal = 5.000.000

sedekah = 1/4

Dit = Sisa uang untuk anak yatim

Jawab = 1/4 x 5.000.000

= 5.000.000/4 = 1.250.000

Uang awal - 1.250.000

5.000.000 - 1.250.000

3.750.000
