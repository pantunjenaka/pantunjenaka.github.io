20. Quiz 15² + 3³ - 15² = ?Pakek caraNote: sendu​


15² + 3³ - 15² =

= (15 × 15) + (3 × 3 × 3) - (15 × 15)

= 225 + 27 - 225

= 252 - 225

= 27

Jawab:

15² + 3³ - 15²

=> (15.15) + (3.3.3) - (15.15)

=> (3.3.3)

=> 27
