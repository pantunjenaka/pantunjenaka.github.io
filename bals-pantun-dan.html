11. | 2X +10 l < 2tolong Di Bals Cepat Kak Pakek Jalan Plizzz


Diketahui:

|2x + 10| < 2


Ditanya:

Himpunan penyelesaian?


PEMBAHASAN

Untuk soal seperti di atas, dapat diselesaikan dengan dua cara.


Cara 1 (Cara Panjang)

|2x + 10|

→ 2x + 10 jika x ≥ -5

→ -2x - 10 jika x < -5


Untuk x ≥ -5

|2x + 10| < 2

2x + 10 < 2

2x < 2 - 10

2x < -8

x < -8/2

x < -4 (memenuhi)


Untuk x < -5

|2x + 10| < 2

-2x - 10 < 2

-2x < 2 + 10

-2x < 12

x > 12/(-2)

x > -6 (memenuhi)


HP = {x | -6 < x < -4, x ∈ R}


Cara 2 (Cara Cepat)

|2x + 10| < 2

-2 < 2x + 10 < 2

-2 - 10 < 2x < 2 - 10

-12 < 2x < -8

-12/2 < x < -8/2

-6 < x < -4


HP = {x | -6 < x < -4, x ∈ R}


Dari kedua cara perhitungan di atas, dapat disimpulkan bahwa himpunan penyelesaian dari |2x + 10| < 2 adalah HP = {x | -6 < x < -4, x ∈ R}
